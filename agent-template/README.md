# Agent Template (shareable starters)

Reusable starter pieces for any new Next.js / TypeScript project. Copy what you need, then customize.

| File                                                   | What it is                                                           |
| ------------------------------------------------------ | -------------------------------------------------------------------- |
| [AGENTS.TEMPLATE.md](./AGENTS.TEMPLATE.md)             | Root `AGENTS.md` skeleton — fill placeholders from the real codebase |
| [eslint-prettier-setup.md](./eslint-prettier-setup.md) | ESLint 9 flat config + Prettier + format/lint-on-save (Tailwind v4)  |

Project-specific API / feature docs stay in `doc/` of each app. Keep **only** cross-project starters here.
